# Game of Thrones survival prediction

## Описание
Бинарная классификация персонажей (жив/мёртв) по данным из вики.  
Модель: логистическая регрессия.

## Файлы
- `train_clean.ipynb` — загрузка, очистка, обучение, сохранение модели.
- `test_clean.ipynb` — загрузка теста, предсказание, `submission.csv`.
